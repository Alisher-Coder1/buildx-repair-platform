# Open Questions

## Purpose

This file stores unresolved questions that must not be hidden inside documents.

## Rules

- Every blocking ambiguity must be written here.
- Each question must have an owner.
- Each question must identify affected documents.
- Questions must be resolved before related documents can become FROZEN.

## Template

```markdown
## OQ-YYYY-MM-DD-001 — <Question Title>

### Question
...

### Context
...

### Affected Documents
- ...

### Decision Needed From
Product Owner / Documentation Architect / Expert

### Status
OPEN / RESOLVED / DEFERRED

### Resolution
...
```
